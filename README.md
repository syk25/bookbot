# bookbot

BookBot is my first [Boot.dev](https://www.boot.dev) project!
Bookbot is a CLI program that counts the words and characters inside a text file.

## How to get started

1. Install Python on your machine.

```bash
brew install python
```

2. Download the code via github
3. Execute the code via terminal with the following command.

```bash
python3 main.py <file_name_path>
```

The `file_name_path` is the path for the text file. I've added some example text file in the books directory. You can test it with the following example.

```bash
python3 main.py books/mobydick.txt
```
